Download ODI dataset from the link provided in 'DataSet Download Link.txt', after downloading rename the odi dataset folder to 'odi_dataset' and keep this in 'Score Prediction Model' folder. Run the model.
After running the model, there will be 2 pkl files generated, don't forget to copy-paste 'XGBRegressor' pkl file in both 'Deploy' folders. One for web and another for Mobile Application. 
In 'Deploy' folder (whichever to test with), run cmd, or powershell, and type 'python app.py' and flask back-end will start to run.
